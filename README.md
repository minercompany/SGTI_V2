# SGTI Enterprise

Sistema de Gestión Integral por Tickets para Cooperativa Reducto.

## Stack Tecnológico
- **Backend**: Java 21 + Spring Boot 3.2
- **Frontend**: Next.js 14 + TypeScript
- **Database**: PostgreSQL 16
- **Cache**: Redis 7
- **Storage**: MinIO
- **Search**: Meilisearch

## Estructura
```
/home/SGTI_V2/
├── backend/          # Spring Boot API
├── frontend/         # Next.js App
├── docs/             # Documentación
└── docker/           # Docker configs
```

## Inicio Rápido
```bash
docker-compose up -d
```
